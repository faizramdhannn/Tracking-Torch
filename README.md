# MULTI-COURIER-API

A lightweight and extensible multi-courier tracking API that allows users to check shipment status from multiple logistics providers using a unified response format.

Currently supported couriers:
- Lion Parcel (Indonesia)
- SiCepat (Indonesia)

Designed with a modular architecture so new couriers can be added easily through separate adapters.

---

## Features

- Unified endpoint for tracking multiple couriers  
- Consistent and clean JSON response  
- Modular adapter-based architecture  
- Customizable error handling  
- Easy to extend for additional couriers  
- Simple and maintainable code structure
